# FDインナー交・調整について

ココでは，FD(フロントディレイラー)のワイヤーを交換したときの調整方法についてまとめる．基本的にはフロントダブルのロードバイクを想定しているが，フロントトリプルのクロスバイクについても後述するつもり．

## ざっくり手順(フロントダブル)
(0. インナーワイヤーをガイドに沿って通してFDにかしめるところまで行ったものとする)
1. インナー側ボルトをしめる(内側プレートが内側チェーンリングに乗るくらい...？)
2. ワイヤーを引っ張りながらかしめる(引っ張りすぎて張りが強くなってしまう可能性もあるが，いったん...)
3. インナー側ボルトをゆるめる(**RDは一番ロー**にいれておき，チェーンがFD内側プレートこ擦らない程度まで)
4. アウターに変速してみる
    1. そもそも張りがゆるすぎてあがらないなら，もういっかいやり直す
    2. ワイヤーの張りは**トリムの可動具合**で確認する
5. アウターの可動域はRD側をハイ側にした時にすらないように
6. STIレバーをさらにアウター側に入れようとしたときに，FDが動く場合はワイヤーがゆるい<br>
    -> アジャスターで調整してみる(インナーに落としてから回すとやりやすい)


## ざっくり手順(フロントトリプル)
1. インナーローで可動域確認
2. RDは最ハイにしておく
3. FD真ん中にしてワイヤー張り具合確認
    1. 外プレートにあたるなら，緩すぎ
    2. 内プレートにあたるなら，張りすぎ
4. アウターハイで可動域確認

フロントダブルに比べると難易度は低いという印象である．

## メモ程度のまとめ(発展途上)
- STIレバー側のワイヤーを確認する．タイコがしっかりハマっているか，ワイヤーがガイド上に沿っているか．
- FDによってワイヤーのかしめ方が特殊な場合がある(最初に確認しておく)．
- ワイヤーの貼り具合はトリムの動きで確認．
- アウター側でさらにSTIレバーでワイヤーを張ろうとしたときにFDが動かないこと(動くようなら張りが弱い)．
- ボルトをいじったときにFD可動域が変わらないときはワイヤーが張りすぎの可能性．
